# 🐱 Gerador de Imagens de Gatinhos

Projeto simples para praticar consumo de API, que gera uma nova imagem de gatinho a cada clique.

## 🚀 Funcionalidades

* Buscar imagens de gatos via API
* Atualizar imagem dinamicamente
* Interface simples e interativa

## 🛠️ Tecnologias utilizadas

* HTML / JavaScript
* API pública de imagens de gatos

## 💡 Melhorias futuras

* Salvar imagens favoritas
* Adicionar loading
* Melhorar interface


## 📸 Como funciona

1. O usuário clica no botão
2. Uma requisição é feita para a API
3. A imagem retornada é exibida na tela

## 🎯 Objetivo

Praticar consumo de APIs e manipulação de dados em aplicações reais.
